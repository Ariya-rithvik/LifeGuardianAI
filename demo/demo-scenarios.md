# Demo Scenarios

## Scenario 1: Elder Fall at Home

1. Fall detected (82% confidence)
2. System enters ALERT
3. Voice: "Are you okay?"
4. No response
5. System enters WARNING
6. Voice: "Please wave if you're okay"
7. No response
8. System enters EMERGENCY
9. Emergency contact notified
10. Emergency services called

## Scenario 2: Child Holding Knife

1. Knife detected in child's hand
2. System enters ALERT
3. Voice: "Please put the object down"
4. No response
5. Parent notified immediately
6. Live camera access shared
